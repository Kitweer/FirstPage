Links

Knowing how to link your code to other document is important

Absolut references

Absolut references (sometimes called external links) connect to pages in other domains. In the href attribute of the tags the full URL of the page, image ot other resources you would like to link to is specified. Once example might be your quest to list some of the greates bakeries of all time. Since you want to link to the bakeries themselves and not to files that you wrote, you would use an absolute reference.

  • Washtenaw Dairy
  • The Dexter Bakery

The code I used to create the link is: 

                    <a href = "https://washtenawdiry.com/" >Washtenaw Dairy</a>
                    <a href = "http://thedexterbakery.com/"> The Dexter Baker</a>

Relative reference

Relative refernces connect to pages within your domain. The href attribute of the tag only the folder (if any) and filename is specified. Some examples of when you might use relative references would be in your navigation bar or links to your own image.
  • Code from this week's lecture on FontAwesome lecture
  • A picture from a subfolder called images.
  • Code from previous week's lecture

The code I used to create the link is: 

                    <a href = "fontawesome.html">Code from this week's lectur on FontAwesome lecture.></a>
